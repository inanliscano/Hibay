HTML Headings

Headings define the structure of a page. Example:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Subheading</h3>

HTML Paragraphs

Use paragraphs to organize content. Example:

<p>This is a paragraph.</p>

HTML Links

Links connect pages. Example:

<a href="https://example.com">Visit Example</a>
